# TrustMark - Universal Watermarking for Arbitrary Resolution Images

A demonstration of embedding a watermark inside an image, and signing the watermark ID into a C2PA manifest using the CAI python SDK

### Dependencies

Run `pip install trustmark` or `pip install .` from within the python folder in this repo to install the TrustMark package and also run `pip install c2pa-python` to install the CAI python SDK upon which this example relies for manifest embedding.
